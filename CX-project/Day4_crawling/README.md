### 피드백 사항

1. 페르소나 - 솔루션 순서 없음
    도출 중 나온 아이디어나 솔루션 다 기록해둘 것

2. 지갑을 열게 할 정도의 페인포인트를 찾아야할 것

3. 하드웨어 VS 소프트웨어
    하드웨어 : 더 혹하게 됨, 그러나 구현과 비용에 한계 > 리스크 대비, 예상 비용 첨부
    소프트웨어 : 사용 시나리오가 아주 구체적이고 시각자료가 개많아야댐

4. 커뮤니티 자체로는 돈을 벌 수 없음. 걍 도구일 뿐

5. 정량적 kpi 파악 필수

6. 대기업은 새시장을 공략하지 않음

7. "기꺼이 돈을 지불할" > 수치적, 정량화

8. 서베이 평균은 30명 이상, 통계적 유의미성을 가질 수 있는 샘플링 수, 심층 인터뷰 있으면 설득력 상승
